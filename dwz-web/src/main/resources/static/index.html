DWZ-JUI Spring Boot 后端服务

这是为 DWZ-JUI 框架练习提供的 Spring Boot 后端服务。

可用的 API 接口:

  • GET /dwz/api/hello - 测试接口
  • GET 数据库连接测试 - 测试数据库连接
  • GET /dwz/api/users - 获取所有用户列表
  • GET /dwz/api/users/{id} - 获取指定ID的用户
  • POST /dwz/api/users - 创建新用户
  • PUT /dwz/api/users/{id} - 更新指定ID的用户
  • DELETE /dwz/api/users/{id} - 删除指定ID的用户

下一步:

请确保你已经启动了 Spring Boot 应用程序，然后你可以：

  1. 将你的 DWZ-JUI 前端文件放置在 static 目录下
  2. 或者从单独的前端服务器访问这些 API

要启动后端服务，请在项目根目录执行以下命令：

mvn spring-boot:run
